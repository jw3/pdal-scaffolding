PDAL Scaffolding
===

Using CMake to scaffold PDAL plugins

### Supported plugins

### Options

- `PLUGIN_TYPE`
- `PLUGIN_NAME`

### Usage

`cmake -DPLUGIN_TYPE=filter -DPLUGIN_NAME=MyFilter <source-path>`
